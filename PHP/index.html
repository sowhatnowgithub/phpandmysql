Tutorial for php setup

First u have to install php on to the local mne

arch -arm64 brew install php // this commands install php in mac

arch -arm64 brew install composer // this commands install package manager for php

Now install visual studio code

Install the following extensions on vs code

  • php server : this install nessary server side handling for the php compilation
  • PHP Intelephense : this is the tool box like auto complertion
  • live server

Now install live server extension for the web browser in chrome web server

  • In the dashboard of the live server extension there two spots to be filled
  • In the actual server Address box, include the address displayed when PHP server extension of the vscode is made on
  • In the live server address box, add the address when live server of the vscode is on and opened in web browser
